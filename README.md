# Mini JS Runtime in Python

This project is built in Python and uses QuickJS Python bindings to execute JavaScript code.  
It accepts JavaScript code from a `.js` file, executes it, and prints the output to stdout.

## Features

- Runs JavaScript code from file
- Supports console.log
- Supports variables, loops, functions, arrays, strings, Math object, spread syntax, etc.
- Built using Python and QuickJS runtime binding
